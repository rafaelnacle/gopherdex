# Gopherdex

This is a guided project from Boot.dev course on Go

### Usage

Clone the project:
```bash
git clone https://github.com/rafaelnacle
```

Build and run the project:
```bash
go build && ./gopherdex
```
If you are on Windows:
```bash
go build && ./gopherdex.exe
```

To see the available commands just type `help` when the project starts on the terminal
